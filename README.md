# Financial Reporting Automation

Python-based automation system for multi-entity financial consolidation and report generation. Designed for FP&A teams working with international operations requiring multi-currency consolidation and standardized Excel outputs.

## 🎯 Purpose

Automates the monthly financial reporting process by:
- Consolidating data from multiple entities/countries
- Handling multi-currency conversion
- Generating standardized Excel reports
- Preparing data for Power BI dashboards
- Reducing manual work from 8+ hours to ~5 minutes

## 🛠️ Technologies

- **Python 3.9+**
- **Pandas** (data manipulation)
- **OpenPyXL** (Excel generation)
- **Requests** (API integration - when applicable)
